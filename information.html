Information Management

Essential Employability Skill – Artifact & Reflection

Artifact — Employee Records Database

📊 Download Records (Excel)

Description: A contract employee qualifications database created for Employment Professionals. The spreadsheet manages records for 30+ employees, tracking Employee IDs, names, pay codes (A/B/C), hourly rates, software proficiency levels (Word, Excel), system access levels, Sage Accounting certification status, and vacation entitlements. It also includes a lookup function for quick employee retrieval by ID and a pay code reference table.

Reflection

EES Outcome & Artifact

This employee records database demonstrates my ability to gather, manage, and organize information effectively. I selected this artifact because it showcases the core skills of information management — structuring data with consistent fields, classifying records with coded systems, and enabling efficient retrieval through lookup functions.

Self-Assessment

I would assess my information management skills at the exemplary level. I designed a structured database with consistent fields, implemented a coded classification system, and built a lookup function for efficient retrieval — all while maintaining accuracy across 30+ records. My prior experience maintaining 98% data accuracy at Divizone Inc. reinforced the precision and organizational habits that made this artifact successful.

Successes

  • Organized 30+ employee records with eight distinct data fields (ID, Name, Pay Code, Rate, Word Level, Excel Level, Access Level, Sage Accounting, Vacation), ensuring each record was complete and consistently formatted
  • Implemented a classification system using pay codes (A, B, C) linked to hourly rates ($30, $25, $18) and vacation entitlements (3, 2, 1 weeks), demonstrating understanding of structured data relationships
  • Built a lookup function enabling quick retrieval of employee information by ID — a critical skill for managing large volumes of records efficiently
  • Maintained accuracy across all entries, drawing on my experience at Divizone Inc. where I managed data entry tasks with 98% accuracy

Challenge

  • Designing the database structure from scratch was initially overwhelming — deciding which fields to include, how to code pay levels, and how to link the lookup function required careful planning. I overcame this by sketching the structure on paper first, mapping out relationships between pay codes, rates, and vacation entitlements before building the spreadsheet, which helped me avoid errors and rework

New Learning & Impact on Future

  • Learned how to design a database structure that balances comprehensive record-keeping with ease of use, including reference tables and lookup tools for efficient data retrieval
  • Developed a deeper understanding of how employee qualifications, certifications, and access levels must be tracked systematically to support organizational decision-making
  • These information management skills are directly applicable to medical office administration where I will manage patient records, track staff credentials, maintain electronic medical records (EMR), and ensure compliance with privacy regulations such as PHIPA
